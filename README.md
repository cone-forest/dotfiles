# Installation
For non-arch systems install packages manually
```bash
curl https://raw.githubusercontent.com/cone-forest/nvim/master/install.sh > /tmp/install.sh && \
chmod +x /tmp/install.sh && \
/tmp/install.sh && \
rm /tmp/install.sh
```
